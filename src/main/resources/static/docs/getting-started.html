서버 실행 확인

이 페이지가 보인다면 서버는 이미 실행 중입니다.

# 아직 실행 안 했다면:
git clone https://github.com/jaeseong90/forma.git
cd forma
./gradlew bootRun

ERP 메인 열기 (admin / 빈 비밀번호) — 이 창은 열어두세요.

STEP 1

YAML로 화면 만들기 3분

부서관리 화면을 YAML 1파일로 만듭니다. Controller, Service, Mapper, HTML — 전부 불필요.

1-1. YAML 파일 생성

design/screens/TUT010.yml 파일을 만드세요:

# design/screens/TUT010.yml
screen:
  id: TUT010
  name: 부서관리
  type: list

auth:
  search: true
  new: true
  save: true
  delete: true
  init: true

search:
  - field: dept_name
    label: 부서명
    widget: text
    placeholder: 부서명 검색

grids:
  grid1:
    editable: true
    checkable: true
    rowNum: true
    columns:
      - { field: DEPT_CODE,   label: 부서코드,   width: 100, editor: text }
      - { field: DEPT_NAME,   label: 부서명,     width: 200, editor: text, required: true }
      - { field: PARENT_CODE, label: 상위부서코드, width: 100, editor: text }
      - { field: DEPT_LEVEL,  label: 레벨,       width: 60,  type: number, align: center }
      - { field: SORT_ORDER,  label: 정렬순서,   width: 70,  editor: text, type: number, align: center }
      - { field: USE_YN,      label: 사용,       width: 60,  editor: check, align: center }

sql:
  tables:
    - { name: tb_dept }
  selectGrid1:
    table: tb_dept
    columns: dept_code, dept_name, parent_code, dept_level, sort_order, use_yn
    orderBy: sort_order, dept_code

1-2. YAML 리로드

서버 재시작 없이 브라우저에서 리로드:

브라우저 주소창에 입력:
http://localhost:8081/api/screen/_reload

→ {"resultCode":"OK","resultData":{"count":6,"screenIds":[...,"TUT010"]}}

1-3. 화면 확인

브라우저 주소창에 입력:
http://localhost:8081/pages/screen.html?id=TUT010

결과

검색폼 + 편집 가능 그리드가 표시됩니다. 조회 버튼(F3)을 누르면 샘플 부서 데이터 10건이 로딩됩니다.
신규(F4) → 행 추가, 편집(더블클릭) → 값 수정, 저장(F9) → DB 반영. YAML 1파일로 완성.

STEP 2

메뉴에 등록하기 2분

만든 화면을 ERP 메인의 좌측 메뉴에 추가합니다.

방법 A: SY010 메뉴관리 화면에서 (권장)

ERP 메인 → 시스템관리 → 메뉴관리(SY010) → 신규 행 추가:

메뉴ID:     M99
메뉴명:     튜토리얼
상위메뉴ID: (비워두면 최상위)
유형:       G (그룹)
정렬순서:   8
사용:       Y

한 행 더 추가:

메뉴ID:     M99A
메뉴명:     부서관리
상위메뉴ID: M99
유형:       P (화면)
프로그램ID: TUT010
URL:        /pages/screen.html?id=TUT010
아이콘:     🏢
정렬순서:   1
사용:       Y

저장(F9) 후 → SY030(역할/권한관리) → ADMIN 역할 클릭 → 부서관리 메뉴의 권한 체크 → 저장

브라우저 새로고침하면 좌측 메뉴에 "튜토리얼 > 부서관리"가 표시됩니다.

3-2. Controller

// src/main/java/com/forma/domain/tutorial/Tut020Controller.java
package com.forma.domain.tutorial;

import com.forma.frame.annotation.*;
import com.forma.frame.base.*;
import com.forma.frame.mybatis.FormaSqlSession;
import lombok.RequiredArgsConstructor;
import org.springframework.web.bind.annotation.*;
import java.util.*;

@FormaController(value = "/tut020", pgmId = "TUT020", description = "공지사항")
@RequiredArgsConstructor
public class Tut020Controller extends BaseController {
    private final Tut020Service service;

    @PostMapping("/selectGrid1")
    public BaseResponse<List<Map<String,Object>>> selectGrid1(
            @RequestBody Map<String,Object> param) {
        return BaseResponse.Ok(service.selectGrid1(param));
    }

    @AddUserInfo
    @PostMapping("/saveGrid1")
    public BaseResponse<?> saveGrid1(@RequestBody List<Map<String,Object>> param) {
        service.saveGrid1(param);
        return BaseResponse.Ok(param);
    }
}

3-3. Service

// src/main/java/com/forma/domain/tutorial/Tut020Service.java
@FormaService(pgmId = "TUT020", description = "공지사항")
public class Tut020Service extends BaseService {
    private final FormaSqlSession sql;
    private final String ns = "tut020";

    public Tut020Service(FormaSqlSession sql) { this.sql = sql; }

    public List<Map<String,Object>> selectGrid1(Map<String,Object> param) {
        return sql.selectList(ns + ".selectGrid1", param);
    }

    @Transactional
    public void saveGrid1(List<Map<String,Object>> param) {
        for (Map<String,Object> item : param) {
            if (Constants.GSTAT_INSERT.equals(item.get(Constants.GSTAT))) {
                sql.insert(ns + ".insertGrid1", item);
            } else {
                sql.update(ns + ".updateGrid1", item);
            }
        }
    }
}

3-4. Mapper XML

<!-- src/main/resources/mapper/domain/tutorial/tut020.xml -->
<mapper namespace="tut020">
    <select id="selectGrid1" parameterType="map" resultType="map">
        SELECT notice_no, title, content, important, view_count,
               created_by, created_at
        FROM tb_notice
        <where>
            <if test="keyword != null and keyword != ''">
                AND (title LIKE '%' || #{keyword} || '%'
                  OR content LIKE '%' || #{keyword} || '%')
            </if>
        </where>
        ORDER BY important DESC, notice_no DESC
    </select>

    <insert id="insertGrid1" parameterType="map">
        INSERT INTO tb_notice (title, content, important,
                               created_by, created_at)
        VALUES (#{title}, #{content}, #{important},
                #{user_id}, NOW())
    </insert>

    <update id="updateGrid1" parameterType="map">
        UPDATE tb_notice
        SET title=#{title}, content=#{content}, important=#{important},
            updated_by=#{user_id}, updated_at=NOW()
        WHERE notice_no = #{notice_no}
    </update>
</mapper>

<div class="forma-page">
  <div class="forma-page-header">
    <h1 class="forma-page-title">공지사항 (TUT020)</h1>
    <div id="forma-toolbar"></div>

<script>
(function() {
    var PGM = 'TUT020';
    var listener = platform.initListener(PGM);
    var ctx = {};

    listener.initPgm = function() {
        ctx.searchForm = new FormaForm('#search-area', {
            search: true,
            elements: [
                { field: 'keyword', label: '검색어', placeholder: '제목/내용' }
            ]
        });
        ctx.grid1 = new FormaGrid('#grid-area', {
            editable: true, checkable: true,
            columns: [
                { field: 'NOTICE_NO', label: '번호', width: 60, readOnly: true, align: 'center' },
                { field: 'TITLE', label: '제목', width: 300, editor: 'text', required: true },
                { field: 'CONTENT', label: '내용', width: 400, editor: 'textarea' },
                { field: 'IMPORTANT', label: '중요', width: 50, editor: 'check', align: 'center' },
                { field: 'VIEW_COUNT', label: '조회수', width: 70, type: 'number', align: 'right' },
                { field: 'CREATED_BY', label: '작성자', width: 80, readOnly: true },
                { field: 'CREATED_AT', label: '작성일', width: 140, readOnly: true }
            ]
        });
        listener.button.search.click();
    };

    listener.button.search.click = function() {
        platform.post('/tut020/selectGrid1', ctx.searchForm.getData(),
            new Callback(function(r) {
                if (r.resultCode === RESULT_CODE.OK) ctx.grid1.setData(r.resultData);
            }));
    };

    listener.button.news.click = function() {
        ctx.grid1.addRow({ IMPORTANT: 'N' });
    };

    listener.button.save.click = async function() {
        var data = ctx.grid1.getCheckedData();
        if (!data.length) { FormaPopup.alert.show('저장할 항목을 선택하세요.'); return; }
        if (!ctx.grid1.checkGridValidation()) return;
        var ok = await FormaPopup.confirm.show('저장하시겠습니까?');
        if (ok) {
            await platform.post('/tut020/saveGrid1', data);
            FormaPopup.toast.success('저장되었습니다.');
            listener.button.search.click();
        }
    };

    listener.button.init.click = function() {
        ctx.searchForm.clear();
        ctx.grid1.clearData();
    };

    if (document.readyState === 'loading') {
        document.addEventListener('DOMContentLoaded', function() { platform.startPage(PGM, listener); });
    } else { platform.startPage(PGM, listener); }
})();
</script>
서버 재시작(./gradlew bootRun) 후 확인. 커스텀 코드는 YAML과 달리 핫 리로드가 안 됩니다.

결과

공지사항 CRUD가 동작합니다. @FormaService 덕분에 감사 로그 + 서비스 트레이스가 자동 기록됩니다.
내용(CONTENT) 컬럼은 editor: 'textarea'로 줄바꿈이 보존되고 높이가 자동 조정됩니다.

STEP 5

기존 DB 테이블 연동 3분

이미 있는 테이블에 YAML을 연결하는 방법입니다.

# 기존 테이블: tb_product (prod_cd, prod_nm, category, price, stock_qty)

screen:
  id: PRD010
  name: 제품관리
  type: list

search:
  - { field: prod_nm, label: 제품명, widget: text }
  - { field: category, label: 카테고리, widget: combo,
      options: [{ value: A, label: 전자 }, { value: B, label: 기계 }] }

grids:
  grid1:
    editable: true
    checkable: true
    columns:
      - { field: PROD_CD,   label: 제품코드, width: 100 }
      - { field: PROD_NM,   label: 제품명,   width: 200, editor: text }
      - { field: CATEGORY,  label: 카테고리, width: 80,  editor: select,
          options: [{ value: A, label: 전자 }, { value: B, label: 기계 }] }
      - { field: PRICE,     label: 단가,     width: 110, editor: currency }
      - { field: STOCK_QTY, label: 재고,     width: 80,  type: number, align: right }

sql:
  tables:
    - { name: tb_product }
  selectGrid1:
    table: tb_product
    columns: prod_cd, prod_nm, category, price, stock_qty
    orderBy: prod_cd
  updateGrid1:
    table: tb_product
    where: "prod_cd = #{prod_cd}"

주의: H2는 resultType=map에서 컬럼명을 대문자로 반환합니다. 그리드 컬럼 field는 대문자로 지정하세요. (PROD_CD, 아니고 prod_cd X)
