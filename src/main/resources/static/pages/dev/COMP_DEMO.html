FormaTab / FormaModal / FormaPopup 데모

Grid 데모 | Form 데모 | 홈
FormaTab API
생성자 옵션타입설명
tabsArray[{ id, label }] 탭 정의
listenerObjectplatform listener (tabBar.tabChange 연동)
onTabChangeFunction(tabId) 탭 변경 콜백
메서드설명
selectTab(tabId) / setActiveTab(tabId)탭 전환
getActiveTab()현재 활성 탭 ID
getContentEl(tabId)탭 패널 DOM (컴포넌트 삽입용)
hideTab(tabId) / showTab(tabId)탭 버튼 숨김/표시
FormaModal API
생성자 옵션타입설명
titleString팝업 제목
urlString팝업 HTML 경로 (fetch로 로딩)
width / heightNumber팝업 크기 (px)
okCallbackFunction(result) 확인 콜백
cancelCallbackFunction(result) 취소 콜백
메서드설명
show(param?)팝업 표시 (param은 팝업 내부에서 접근 가능)
ok(result)결과 전달 후 닫기 → okCallback 호출
cancel(result?)취소 후 닫기 → cancelCallback 호출
SIZE 프리셋크기
FormaModal.SIZE.XS400 × 300
FormaModal.SIZE.S600 × 400
FormaModal.SIZE.M800 × 600
FormaModal.SIZE.L1000 × 700
FormaModal.SIZE.XL1200 × 800
FormaPopup API
메서드설명
FormaPopup.alert.show(msg)알림 (Promise, 확인 버튼)
FormaPopup.confirm.show(msg, callback)확인/취소 (callback(true/false))
FormaPopup.loading.show() / .hide()로딩 오버레이
FormaPopup.toast.success(msg)성공 토스트 (3초)
FormaPopup.toast.error(msg)에러 토스트
FormaPopup.toast.info(msg)정보 토스트
1. 기본 탭 (FormaTab)
탭 클릭으로 패널 전환. onTabChange 콜백. 각 탭에 폼/그리드 배치 가능.
2. 탭 동적 제어 (hideTab / showTab)
런타임에 탭 숨기기/표시. 권한에 따라 탭 제어 시 활용.
3. 좌우 스플릿 + 우측 탭 (ERP 복합 레이아웃)
좌측 목록 그리드 | 우측 탭(기본정보 폼 + 품목 그리드). 행 클릭 시 우측 연동. SDB010 패턴.
4. FormaPopup — Alert / Confirm / Toast / Loading
다이얼로그, 확인/취소, 토스트 메시지, 로딩 오버레이.
5. FormaModal — 모달 팝업 (인라인 HTML)
FormaModal로 팝업 생성. SIZE 프리셋(XS~XL). OK/Cancel 콜백. ESC/오버레이 클릭 닫기.
6. 모달 팝업에서 그리드 선택 → 부모 반영
팝업 열기 → 그리드에서 행 선택 → OK → 부모 폼에 값 반영. codePopup 패턴.
