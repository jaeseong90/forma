1. 셀 병합 (merge: true) — 자동 rowspan
같은 값이 연속된 행의 셀을 자동으로 병합(rowspan). 거래처, 상태 컬럼에 merge: true 적용. mergeCells() API로 수동 병합도 가능.
2. 멀티행 레코드 (rowDetail) — 1 레코드 = N행, 편집 가능
각 레코드 아래에 서브행 표시. 배송지/연락처/비고 더블클릭으로 인라인 편집 가능. editable 기본 활성.
3. 정렬 (기본 활성) + 행 드래그&드롭 재정렬
헤더 클릭: asc→desc→해제 (정렬은 기본 활성). 좌측 ≡ 핸들 드래그로 행 순서 변경. onRowReorder 콜백 지원.
4. 기본 에디터 (text / number)
더블클릭 또는 Enter/F2로 편집. Tab으로 다음 셀 이동. gstat 자동 관리(I=신규, U=수정).
5. select / combo 에디터
select: 기본 HTML select. combo: 검색 가능 드롭다운 (대량 옵션에 적합). 더블클릭으로 편집.
6. check / switch 에디터
check: 체크박스 (항상 표시). switch: 토글 스위치 (항상 표시). 클릭으로 즉시 전환. Y/N 값.
7. date 에디터
더블클릭 시 커스텀 캘린더 팝업. 날짜 선택/초기화 지원. YYYY-MM-DD 형식.
8. currency 에디터
셀 표시: 천단위 콤마. 더블클릭 편집: 순수 숫자 입력. 숫자/백스페이스/마이너스/소수점만 허용. footer: sum/avg/count.
9. textarea 에디터
더블클릭 시 멀티라인 입력 영역 표시. 긴 텍스트 편집에 적합. Esc로 취소.
10. 셀 렌더러 (renderer) + 필터행 (filterable)
커스텀 HTML 렌더링: 뱃지, 프로그레스바, 링크, 버튼. 헤더 아래 필터 입력으로 실시간 필터링.
11. 복합 그리드 (모든 에디터 + 키보드 + 클립보드)
모든 에디터 타입 혼합. 키보드 내비게이션(Arrow/Tab/Enter/F2). Ctrl+C 복사 / Ctrl+V 붙여넣기. filterable 필터행.
12. 정렬 + 페이징 (대량 데이터)
헤더 클릭으로 정렬 (asc → desc → 해제). 페이징 바 (20/50/100건). 100건 샘플 데이터.
13. 가상 스크롤 (virtualScroll) — 10,000건
virtualScroll: true, height: 350. DOM에 보이는 행만 렌더링하여 대량 데이터에서도 부드러운 스크롤. 스크롤 후 편집/정렬 가능.
14. 행 그룹핑 (groupBy) + 소계 (groupFooter)
groupBy: '상태' 필드로 그룹핑. 그룹 헤더 클릭으로 접기/펼치기. 그룹별 소계 행 표시.
15. 트리 그리드 (treeField) — 계층형 데이터
treeField로 트리 표시. id/parent_id 기반 계층. ▼/▶ 접기/펼치기. 편집 가능.
16. 마스터-디테일 중첩 그리드 (detailGrid)
▶ 클릭 시 하위 그리드 펼침. 독립 편집/정렬. XLSX 내보내기 가능.
17. N단 멀티헤더 (3단) + frozen 컬럼
3단 헤더: 반기 → 분기 → 월. 좌측 거래처 2컬럼 고정(frozen). rowspan 자동 계산. 수평 스크롤 확인.
16. N단 멀티헤더 (4단) — 상용 그리드 수준
4단 헤더: 전체 → 지역 → 분류 → 항목. 중간 레벨 colspan/rowspan 자동 조합. 단일 컬럼은 4행 rowspan.
19. 스플릿 레이아웃 — 좌우 분할 (그리드 + 폼/탭)
좌측 그리드 | 우측 입력폼. 핸들 드래그로 너비 조정. ◀ 버튼으로 접기/펼치기. 중첩 가능. minSize: 200px.
상세정보 (우측 패널)
20. 상하 분할 — 마스터 그리드 + 디테일 그리드
상단 그리드 | 하단 그리드. 핸들 드래그로 높이 조정. 접기/펼치기 지원.
21. 이벤트 (onRowClick, onCellChange) + 셀 CSS
행 클릭 / 셀 값 변경 시 로그 출력. 수량×단가=금액 자동계산. 음수 금액은 빨간색 표시.
