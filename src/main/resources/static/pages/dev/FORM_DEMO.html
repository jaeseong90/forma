1. 검색폼 (search: true)
가로 배치, 콤보/날짜범위 포함
2. 입력폼 기본 (2컬럼, columns: 2)
text, number, combo, date, textarea
3. 3컬럼 폼 + colspan (columns: 3)
비고 필드가 3칸 차지, labelWidth: 100
4. 1컬럼 폼 (columns: 1)
한 줄씩 세로 나열, labelWidth: 120
5. 전체 위젯 (4컬럼, columns: 4)
text, number, combo, date, dateRange, checkbox, radio, textarea, codePopup, hidden
6. Validation 데모 (required 검증)
이름, 이메일 필수. validate() 클릭 시 빈 필드 에러 표시
7. getField / setField 데모
개별 필드 읽기/쓰기
8. Currency (금액 입력, prefix/suffix)
콤마 자동 포맷, 포커스 시 순수숫자, getData()에서 Number 반환. prefix/suffix 표시.
9. MultiCombo (복수 선택)
체크박스 드롭다운, pill 태그, 전체선택/해제. getData()에서 쉼표 구분 문자열.
10. YearMonth / Year (년월/년도 선택)
yearMonth: YYYY-MM, year: ±5년 드롭다운
11. Switch (ON/OFF 토글)
슬라이드 토글, Y/N 반환. onLabel/offLabel 지원.
12. Divider + Display / Badge
구분선, 읽기전용 텍스트, 상태 뱃지
13. 의존 콤보 (dependsOn)
상위 콤보(품목그룹) 변경 → 하위 콤보(품목) 옵션 갱신. (데모에서는 직접 옵션 교체)
14. 이벤트 (onChange, onBlur, onEnter)
값 변경 시 로그 출력. 수량×단가=금액 자동 계산.
15. 변경 감지 (isDirty / getChangedFields)
setData로 원본 세팅 후 값 변경하면 isDirty() = true
16. Password + 동적 제어 (setRequired, setReadonly, setLabel, focus)
비밀번호 보기/숨기기, 동적으로 필수/읽기전용/라벨 변경
17. Time / File Upload
time: 시간 선택 (HH:MM). file: 파일 업로드 (accept, multiple 지원).
18. Collapsible Divider (접기/펼치기 구분선)
collapsible: true 옵션으로 섹션 접기/펼치기. 클릭 시 다음 divider 까지의 필드 숨김/표시.
19. Autocomplete (자동완성 검색)
타이핑 시 옵션 필터링. 정적 옵션 또는 fetchUrl로 서버 검색 지원. 키보드 Arrow/Enter 선택.
