Grid | Form | Tab/Modal | AI 가이드 | 홈
1. 테마 시스템 (FormaTheme)
CSS 변수 기반. light/dark/blue 3종. localStorage 저장. 커스텀 테마 추가 가능.
텍스트   Primary   Danger   Success   Warning  
2. 다국어 시스템 (FormaI18n)
ko/en/ja/zh 4개 언어. FormaI18n.t('key') 번역. addMessages()로 커스텀 메시지 추가.
3. 차트 (FormaChart) — Bar / Line / Pie / Donut / HBar
순수 SVG 차트. 외부 라이브러리 없음. 애니메이션, 툴팁, 범례 지원.
4. 메뉴 시스템 (FormaMenu) — 좌측 트리 + MDI 탭
트리형 메뉴. 클릭 시 iframe 탭으로 열림. 다중 탭 MDI. 탭 닫기 지원.
5. Excel 임포트 (importExcel) + 인쇄 (print)
CSV/TSV 파일 업로드 → 그리드 데이터 추가. 인쇄 버튼으로 새 창 인쇄.
