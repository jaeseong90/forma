FormaGrid 주요 기능

  • 에디터 8종: text, select, combo, date, currency, check, switch, textarea
  • N단 멀티헤더 (무제한 colspan/rowspan)
  • 셀 병합 (merge: true, mergeCells API)
  • 멀티행 레코드 (rowDetail, 편집 가능)
  • 트리 그리드 (treeField, 접기/펼치기)
  • 마스터-디테일 중첩 그리드 (detailGrid)
  • 가상 스크롤 (10,000건+)
  • 행 그룹핑 + 소계 (groupBy, groupFooter)
  • 키보드 셀 내비게이션 (16키)
  • 클립보드 Ctrl+C/V (Excel 호환)
  • Undo/Redo (Ctrl+Z/Y, 100건 스택)
  • 우클릭 컨텍스트 메뉴 (헤더/셀)
  • 컬럼 숨기기/표시, 자동맞춤
  • 행 드래그&드롭 재정렬
  • 필터행 (텍스트/select)
  • 커스텀 셀 렌더러 (HTML/DOM)
  • frozen 컬럼, footer 집계, 페이징
  • CSV + XLSX 내보내기 (서식 포함)
  • gstat 자동관리, 검증, 셀CSS, 툴팁

FormaForm 위젯 21종

  • text, textarea, password, hidden
  • combo (검색 드롭다운), multiCombo (복수 선택)
  • autocomplete (자동완성 검색)
  • date, dateRange (캘린더+프리셋), time
  • yearMonth, year
  • currency (금액 콤마포맷)
  • checkbox, radio, switch (ON/OFF)
  • codePopup (팝업 검색)
  • file (파일 업로드, accept/multiple)
  • display (읽기전용/뱃지)
  • divider (구분선, 접기/펼치기)
  • ― API ―
  • getData/setData, getField/setField
  • validate, formReadonly, isDirty
  • setOptions, setRequired, setReadonly
  • showField/hideField, focus
  • getChangedFields, resetToOriginal
  • destroy (메모리 정리)
