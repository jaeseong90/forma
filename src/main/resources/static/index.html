FORMA

YAML 1파일로 ERP 화면을 완성하는 AI 시대의 업무시스템 개발 프레임워크

ERP 데모 시작 GitHub
v0.2.0 · Spring Boot 3.4 · Java 17

Quick Start

# 1. 클론
git clone https://github.com/jaeseong90/forma.git
cd forma

# 2. 실행 (Java 17 필요)
./gradlew bootRun

# 3. 브라우저
http://localhost:8081 ← 이 페이지
http://localhost:8081/login.html ← ERP 메인 (admin / 빈 비밀번호)

Features

📄

YAML = 화면

Controller, Service, Mapper, HTML 없이 YAML 1파일만 작성하면 완전한 CRUD 화면이 런타임에 생성됩니다.

🖥️

SPA MDI

iframe 없는 div 기반 MDI. 메모리 효율적이고 화면간 직접 통신이 가능합니다.

🔐

3계층 권한

역할-메뉴-데이터 권한. 버튼 단위 접근 제어와 부서/사용자 단위 데이터 필터링.

📊

ERP 그리드

인라인 편집 8종, 셀 병합, 트리, 가상 스크롤, 그룹핑, 키보드, 클립보드, Undo.

📝

21종 폼 위젯

text, combo, date, currency, codePopup, switch, multiCombo 등 ERP 실무에 필요한 위젯.

📋

감사 로그

INSERT/UPDATE/DELETE 자동 추적. Before/After JSON 비교. 서비스 실행시간 트레이스.

YAML 1파일 = 완성 화면

screen:
  id: MMA010
  name: 품목관리
  type: list

search:
  - field: item_nm
    label: 품목명
    widget: text

grids:
  grid1:
    editable: true
    columns:
      - { field: ITEM_CD, label: 품목코드 }
      - { field: ITEM_NM, label: 품목명 }
      - { field: UNIT_PRICE, label: 단가 }

sql:
  tables:
    - { name: tb_item }

이 YAML 파일 하나로:

  • 검색폼 자동 생성
  • 데이터 그리드 자동 생성
  • 조회 SQL 동적 생성
  • INSERT/UPDATE SQL 동적 생성
  • DELETE SQL 동적 생성
  • 서버 페이징 지원
  • 데이터 권한 자동 적용
  • 감사 로그 자동 기록

Controller, Service, Mapper XML, HTML — 전부 불필요

지원 레이아웃

list

검색폼 + 편집 가능 그리드. 가장 기본적인 CRUD 화면.

예: 거래처관리, 품목관리

split-detail

좌측 마스터 그리드 클릭 → 우측 디테일 그리드 로딩. 좌우 분할.

예: 견적템플릿관리

master-detail

상단 마스터 그리드 + 중간 입력폼 + 하단 디테일 그리드.

예: 수주관리
