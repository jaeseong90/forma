# 프론트엔드 컴포넌트 구조

## 개요

FORMA 프론트엔드는 빌드 도구 없이 Vanilla JS로 구성된다.
`/static/forma/forma.js` 하나에 모든 공통 컴포넌트가 포함되어 있으며,
화면별 JS는 이 컴포넌트들을 조합하여 구성한다.

## 컴포넌트 목록

### FormaApi - API 통신

fetch 기반 HTTP 클라이언트. 모든 응답에서 `success` 필드를 검사하여 에러 시 자동으로 throw한다.

| 메서드 | 설명 |
|--------|------|
| `FormaApi.get(url, params)` | GET 요청. params는 URLSearchParams로 변환 |
| `FormaApi.post(url, body)` | POST 요청. body는 JSON 직렬화 |
| `FormaApi.del(url)` | DELETE 요청 |

### FormaGrid - 데이터 그리드

테이블 형태의 데이터 그리드. 읽기 전용 및 편집 모드를 지원한다.

**생성자 옵션:**

```javascript
new FormaGrid('#grid', {
    columns: [
        { field: 'cust_cd', label: '거래처코드', width: 120 },
        { field: 'status', label: '상태', format: 'badge', code: 'ORD_STATUS' },
        { field: 'total_amt', label: '합계', format: 'currency', align: 'right' },
        { field: 'seq', label: '순번', auto: true, readOnly: true },
    ],
    editable: false,           // true면 셀 편집 가능
    onSelect: (row, idx) => {},  // 행 선택 콜백
    onCellChange: (row, field, idx) => {},  // 셀 변경 콜백 (editable시)
    features: ['addRow', 'deleteRow'],      // 편집 도구 버튼
});
```

**컬럼 속성:**

| 속성 | 설명 |
|------|------|
| `field` | 데이터 필드명 |
| `label` | 헤더 표시명 |
| `width` | 컬럼 너비 (px) |
| `format` | 포맷 적용 (`currency`, `number`, `date`, `badge`) |
| `code` | 코드 그룹명 (badge 포맷 시 필수) |
| `align` | 텍스트 정렬 (`left`, `center`, `right`) |
| `readOnly` | 편집 불가 (editable 모드에서도) |
| `auto` | 자동 순번 (편집 불가) |
| `link` | 클릭 시 이동할 화면ID 또는 함수 |
| `frozen` | 좌측 고정 컬럼 |

**주요 메서드:**

| 메서드 | 설명 |
|--------|------|
| `setData(rows, total)` | 데이터 설정 및 렌더링 |
| `getSelectedRow()` | 선택된 행 반환 (없으면 null) |
| `getAllRows()` | 전체 행 배열 반환 |
| `addRow()` | 빈 행 추가 (editable 모드) |
| `deleteRow()` | 선택된 행 삭제 (editable 모드) |

### FormaSearch - 검색바

검색 조건 입력 UI. 위젯 타입에 따라 자동으로 입력 요소를 생성한다.

```javascript
new FormaSearch('#search', [
    { field: 'cust_cd', widget: 'text', label: '거래처코드' },
    { field: 'cust_type', widget: 'combo', label: '유형', code: 'CUST_TYPE' },
    { field: 'order_date', widget: 'dateRange', label: '수주일자', default: 'THIS_MONTH' },
], () => page.doSearch());
```

**위젯 타입:**

| widget | 생성 요소 | 파라미터 키 |
|--------|-----------|-------------|
| `text` | `<input type="text">` | `{field}` |
| `combo` | `<select>` (코드 자동 로딩) | `{field}` |
| `dateRange` | `<input type="date">` x 2 | `{field}_from`, `{field}_to` |

**주요 메서드:**

| 메서드 | 설명 |
|--------|------|
| `getValues()` | 입력값을 `{ key: value }` 객체로 반환 (빈 값 제외) |
| `reset()` | 모든 입력 초기화 |

### FormaForm - 입력 폼

등록/수정용 폼 빌더. 필드 정의에 따라 자동으로 입력 요소를 생성한다.

```javascript
new FormaForm('#form', [
    { field: 'cust_cd', label: '거래처코드', required: true },
    { field: 'cust_type', widget: 'combo', label: '유형', code: 'CUST_TYPE' },
    { field: 'remark', widget: 'textarea', label: '비고' },
    { field: 'active', widget: 'checkbox', label: '사용여부' },
]);
```

**위젯 타입:** `text`(기본), `combo`, `textarea`, `checkbox`, `datepicker`

**주요 메서드:**

| 메서드 | 설명 |
|--------|------|
| `getData()` | 폼 데이터를 `{ key: value }` 객체로 반환 |
| `setData(obj)` | 객체로 폼 값 설정 |
| `clear()` | 폼 초기화 |

### FormaPopup - 코드 검색 팝업

모달 팝업으로 코드/거래처 등을 검색하여 선택하는 컴포넌트.

```javascript
FormaPopup.open({
    title: '거래처 검색',
    url: '/api/customer',
    columns: [
        { field: 'cust_cd', label: '코드', width: 100 },
        { field: 'cust_nm', label: '거래처명', width: 200 },
    ],
    onSelect: (row) => {
        // 선택된 행 데이터 처리
        form.inputs['cust_cd'].value = row.cust_cd;
    }
});
```

### FormaPage - 페이지 헬퍼

액션 버튼바 생성 및 UI 유틸리티.

```javascript
FormaPage.actionBar('#actions', [
    { label: '조회', onClick: () => page.doSearch(), primary: true },
    { label: '저장', onClick: () => page.doSave() },
    { label: '삭제', onClick: () => page.doDelete() },
]);

FormaPage.confirm('정말 삭제하시겠습니까?');  // window.confirm 래퍼
FormaPage.alert('저장되었습니다.');            // window.alert 래퍼
```

### FormaFormat - 포맷터

데이터 표시 포맷 변환.

| 포맷 | 설명 | 예시 |
|------|------|------|
| `currency` | 천단위 콤마 (ko-KR) | `1234500` → `1,234,500` |
| `number` | 숫자 포맷 | `1234` → `1,234` |
| `date` | 날짜 (앞 10자리) | `2026-03-17T00:00` → `2026-03-17` |
| `badge` | 코드별 색상 라벨 | `DRAFT` → `<span class="forma-badge">작성중</span>` |

### getCodeItems() - 코드 캐시

코드 사전을 `/api/codes/{group}` API로 로딩하고 클라이언트 측에서 캐싱한다.
FormaSearch의 combo, FormaGrid의 badge에서 내부적으로 사용한다.

## 화면 JS 패턴

모든 화면 JS는 동일한 구조를 따른다:

```javascript
const page = {
    api: '/api/{endpoint}',

    init() {
        // 1. 검색바 생성
        this.search = new FormaSearch('#search', [...], () => this.doSearch());
        // 2. 그리드 생성
        this.grid = new FormaGrid('#grid', { columns: [...] });
        // 3. 버튼바 생성
        FormaPage.actionBar('#actions', [...]);
        // 4. 초기 조회
        this.doSearch();
    },

    async doSearch() {
        const data = await FormaApi.get(this.api, this.search.getValues());
        this.grid.setData(data.data, data.total);
    },

    async doSave() { /* 저장 로직 */ },
    async doDelete() { /* 삭제 로직 */ },
};

document.addEventListener('DOMContentLoaded', () => page.init());
```

## CSS 클래스 체계

모든 CSS 클래스는 `forma-` 접두사를 사용한다.

| 카테고리 | 주요 클래스 |
|----------|-------------|
| 레이아웃 | `forma-page`, `forma-page-title`, `forma-section` |
| 입력 | `forma-input`, `forma-btn`, `forma-btn-primary`, `forma-btn-sm` |
| 검색 | `forma-search`, `forma-search-row`, `forma-search-group` |
| 그리드 | `forma-grid-wrap`, `forma-grid`, `forma-grid-input`, `forma-grid-empty` |
| 폼 | `forma-form`, `forma-form-row` |
| 팝업 | `forma-popup-overlay`, `forma-popup`, `forma-popup-header` |
| 기타 | `forma-badge`, `forma-link`, `forma-action-bar` |
